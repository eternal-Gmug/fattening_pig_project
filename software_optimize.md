# 添加标注框的列表更新
目前现状：
- 当新增一个标注框时，列表会全部重新渲染一遍，造成资源的浪费
改进优化：
- 当新增一个标注框时，只需要在列表的末尾添加一个新项，而不是重新渲染整个列表（idea：使用列表视图的addItem方法）
- 当删除一个标注框时，只需要从列表中移除对应的项，而不是重新渲染整个列表（idea：使用列表视图的removeItem方法）
# 优化视频显示区域的缩放比例
目前现状：
- 视频显示区域是固定大小的，当界面可缩放时，视频显示区域的大小不会改变，如果处理的图像标注框太多，会导致标注人员的操作困难
改变优化：
- 视频显示区域的大小会根据软件窗口的大小进行调整，并保持原视频比例
- 在已有加载图像的基础上，加载的图像也会根据视频显示区域的大小进行缩放，保持原视频比例
